## Why auto-cpufreq?

One of the problems with Linux on laptops is that the CPU will run in an unoptimized manner which negatively impacts battery life. The CPU may use the "performance" governor with turbo boost enabled regardless of whether it's plugged in or not.

auto-cpufreq solves this by automatically making CPU frequency-related changes based on battery state, CPU usage, temperature, and system load — letting the Linux kernel do most of the heavy lifting while providing intelligent optimizations.

## Quick Start

<Steps>
  <Step title="Install auto-cpufreq">
    Clone the repository and run the installer:
    ```bash
    git clone https://github.com/AdnanHodzic/auto-cpufreq.git
    cd auto-cpufreq && sudo ./auto-cpufreq-installer
    ```
  </Step>
  
  <Step title="Install the daemon">
    Enable automatic CPU optimizations:
    ```bash
    sudo auto-cpufreq --install
    ```
  </Step>
  
  <Step title="View stats">
    Monitor the optimizations in real-time:
    ```bash
    auto-cpufreq --stats
    ```
  </Step>
</Steps>

## Supported Devices

auto-cpufreq supports laptops and desktops with:

- **Intel** CPUs (all generations)
- **AMD** CPUs (all generations)  
- **ARM** CPUs

Works on any Linux distribution with a 5.0+ kernel.
