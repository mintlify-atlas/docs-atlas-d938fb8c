auto-cpufreq is a command-line tool that provides automatic CPU speed and power optimization for Linux systems.

## Quick Start

Run auto-cpufreq without any arguments to see the help message:

```bash
auto-cpufreq
```

## Available Commands

auto-cpufreq supports the following command-line options:

### Monitoring & Optimization

- [--monitor](/cli/monitor) - Monitor and see suggestions for CPU optimizations
- [--live](/cli/live) - Monitor and make temporary CPU optimizations
- [--stats](/cli/stats) - View live stats of CPU optimizations made by daemon

### Daemon Management

- [--install](/cli/install) - Install daemon for permanent automatic CPU optimizations
- [--remove](/cli/remove) - Remove daemon and stop automatic CPU optimizations
- [--update](/cli/update) - Update daemon and package to the latest version

### Manual Control

- [--force](/cli/force) - Force use of specific CPU governor (powersave/performance)
- [--turbo](/cli/turbo) - Control CPU turbo mode behavior
- [--config](/cli/config) - Use custom configuration file

### Additional Features

- [--bluetooth](/cli/bluetooth) - Control Bluetooth boot behavior
- [--debug](/cli/debug) - Show debug information for troubleshooting
- [--version](/cli/version) - Show currently installed version

## Common Usage Patterns

### First-time Setup

1. Monitor your system to see what optimizations would be applied:
   ```bash
   sudo auto-cpufreq --monitor
   ```

2. Test live optimizations temporarily:
   ```bash
   sudo auto-cpufreq --live
   ```

3. Install the daemon for permanent optimizations:
   ```bash
   sudo auto-cpufreq --install
   ```

### Monitoring and Stats

Check current optimization statistics:
```bash
auto-cpufreq --stats
```

### Troubleshooting

Generate debug information for bug reports:
```bash
sudo auto-cpufreq --debug
```

## Root Privileges

Most auto-cpufreq commands require root privileges (sudo) because they need to:
- Read system power and frequency information
- Modify CPU governor settings
- Install or manage system services
- Configure hardware settings

Commands that don't require root:
- `--version`
- `--help`
- `--donate`
